# 🚀 Karan Singh Bohra | Full Stack Portfolio

Welcome to my professional portfolio! This project is a highly responsive, modern web application built to showcase my journey as a Full Stack Developer, highlighting my technical skills, certifications, and real-world projects.

## ✨ Key Features
- **Modern UI/UX**: Clean, dark-themed design with premium indigo accents.
- **Dynamic Content**: Data-driven components powered by React and a centralized constant file.
- **Project Showcase**: Detailed project cards with integrated image galleries and live/source links.
- **Experience & Education**: Timeline-based visualization of my professional and academic background.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Interactive Elements**: Smooth scrolling, hover effects, and modern transitions.

## 🛠️ Tech Stack
- **Frontend:** React.js (v19), Vite
- **Styling:** Tailwind CSS (Custom Design System)
- **Icons:** Lucide React
- **Language:** TypeScript
- **Deployment:** Vercel / GitHub Pages

## 📁 Project Structure
```bash
├── components/        # Reusable React components
├── public/            # Static assets (images, PDFs)
├── App.tsx            # Main Application logic & routing
├── constants.tsx      # Centralized data store
├── types.ts           # TypeScript interfaces
└── index.html         # Entry point
```

## ⚙️ Installation & Setup

1. **Clone the repo:**
   ```bash
   git clone https://github.com/bohrakaran/portfolio.git
   ```

2. **Navigate to directory:**
   ```bash
   cd portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start Development Server:**
   ```bash
   npm run dev
   ```
